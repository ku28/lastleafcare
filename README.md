## How to install

1. Clone this repositoy:

```bash
git clone https://github.com/ku28/lastleafcare.git
```

2. Go into project

```bash
cd lastleafcare
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
